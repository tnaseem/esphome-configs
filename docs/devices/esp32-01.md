# Device: esp32-01

## Summary

| Property | Value |
|----------|-------|
| Hardware | M5Stack AtomS3 Lite (ESP32-S3) |
| IP address | 192.168.1.22 |
| Current role | Stairlift control |
| Role file | `roles/stairlift-control.yaml` |

## Purpose

`esp32-01` controls a motorised stairlift via IR transmission. It exposes two Home Assistant switches (`Stairlift Up` and `Stairlift Down`) that continuously re-transmit the appropriate IR burst code at 35 ms intervals while active.

See [Stairlift Control](../features/stairlift-control.md) for full feature documentation.

## Package Composition

```yaml
packages:
  base: !include common/base.yaml
  wifi: !include common/wifi.yaml
  role: !include roles/stairlift-control.yaml
```

## Flashing

```bash
esphome run esp32-01.yaml --device 192.168.1.22
```

## Changing Role

To assign a different role, update the `role:` line in `esp32-01.yaml`:

```yaml
# Example: switch to offline test mode
role: !include roles/stairlift-test.yaml
```

Then reflash the device.
