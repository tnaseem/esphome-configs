# Device: esp32-02

## Summary

| Property | Value |
|----------|-------|
| Hardware | M5Stack AtomS3 Lite (ESP32-S3) |
| IP address | 192.168.1.23 |
| Current role | Bluetooth proxy |
| Role file | `roles/bluetooth-proxy.yaml` |

## Purpose

`esp32-02` acts as a Bluetooth proxy, extending the Bluetooth range of Home Assistant. It passively bridges BLE advertisements and connections between HA and nearby Bluetooth devices (sensors, locks, etc.) that are out of range of the HA host.

## Package Composition

```yaml
packages:
  base: !include common/base.yaml
  wifi: !include common/wifi.yaml
  role: !include roles/bluetooth-proxy.yaml
```

## Flashing

```bash
esphome run esp32-02.yaml --device 192.168.1.23
```

## Changing Role

To assign a different role, update the `role:` line in `esp32-02.yaml`:

```yaml
# Example: switch to IR receiver for capturing new remote codes
role: !include roles/remote-receiver.yaml
```

Then reflash the device.
